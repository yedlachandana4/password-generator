# 🔐 Password Generator (Python)

## 📌 Description

This is a simple Python project that generates strong and random passwords based on user input.

## 🚀 Features

* Generate secure random passwords
* User can choose password length
* Uses letters, numbers, and special characters

## 🛠️ Technologies Used

* Python
* random module
* string module

## ▶️ How to Run

1. Install Python
2. Open terminal / command prompt
3. Run the file:

   ```bash
   python password_generator.py
   ```

## 📷 Example Output

```
Enter password length: 8  
Generated Password: A@7k#2Lp
```
